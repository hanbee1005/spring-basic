# spring-basic
인프런 강좌 중 김영한 개발자님의 '스프링 핵심 원리 - 기본편'을 수강하며 정리한 내용입니다.
객체지향 언어의 핵심인 다형성과 OCP, DIP를 만족하는 프로젝트를 개발하면서 스프링의 원리를 이해하는 강좌입니다.

## 사용 기술 및 환경
- Java 11
- SpringBoot 2.5.0
- Gradle
